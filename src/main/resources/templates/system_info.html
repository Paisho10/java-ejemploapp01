Implementing Web Templates with Thymeleaf in Spring Boot

  1. Create a Spring Boot Project: Start by creating a new Spring Boot project using your preferred IDE or the Spring Initializr.
  2. Add Thymeleaf Dependency: In your project's pom.xml (Maven) or build.gradle (Gradle), add the Thymeleaf dependency: 
    <dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-thymeleaf</artifactId>
</dependency>
  3. Create Thymeleaf Templates: Create HTML templates using Thymeleaf syntax. Place these templates in the src/main/resources/templates directory. For example, create a file named example.html: 
    <!DOCTYPE html>
<html xmlns:th="http://www.thymeleaf.org">
<head>
    <meta charset="UTF-8">
    <title>My Thymeleaf Template</title>
</head>
<body>
    <h1>Welcome to Thymeleaf</h1>
    <p th:text="${message}"></p>
</body>
</html>
  4. Create a Controller: Create a controller class that maps to a URL and returns the Thymeleaf template. For example: 
    import org.springframework.stereotype.Controller;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.GetMapping;

@Controller
public class ExampleController {

    @GetMapping("/example")
    public String example(Model model) {
        model.addAttribute("message", "Hello, Thymeleaf!");
        return "example";
    }
}
  5. Configure View Resolver: Ensure that your Thymeleaf template resolver and view resolver configurations are correctly set up in your application.properties or application.yml: 
    spring.thymeleaf.prefix=classpath:/templates/
spring.thymeleaf.suffix=.html
  6. Create CSS Stylesheet: Create a CSS file named styles.css and place it in the src/main/resources/static/css directory. You can define your custom styles in this file.
  7. Link the CSS Stylesheet: Link the CSS file in your Thymeleaf templates using the following line within the <head> section: 
    <link rel="stylesheet" type="text/css" href="/css/styles.css">
  8. Start Your Application: Start your Spring Boot application, and you can access the Thymeleaf template at the specified URL, e.g., http://localhost:8080/example.

Controller Information Swagger-like Documentation

InfoController - Swagger-like Documentation

Endpoints

Endpoint HTTP Method Description
/getinfo/list GET Get the list of info items.
/getinfo/showlist GET Get the current list of info items.
/getinfo/list/add POST Add a member to the list.
/getinfo/list/remove POST Remove a member from the list.
/getinfo/queue GET Get the queue of info items.
/getinfo/queue/add POST Add a member to the queue.
/getinfo/queue/remove POST Remove a member from the queue.
